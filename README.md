

Aether & Earth
A modern, full-stack e-commerce platform blending minimalist design with seamless functionality.



About the Project


Aether & Earth is a fully responsive, data-driven clothing and accessories storefront. Built with the MERN stack, it features a premium "glassmorphic" UI, dynamic product filtering, secure JWT-based authentication, and a complete checkout-to-dashboard user flow.

Key Features

Modern UI/UX: Cinematic hero carousels, glassmorphic filter bars, and smooth micro-animations.

Dynamic Catalog: Browse 16 meticulously curated products with real-time category filtering (Men, Women, Kids, Accessories) and price sorting.

State Management: Persistent shopping cart functionality powered by Redux.

Secure Authentication: User login and registration using JSON Web Tokens (JWT) and bcrypt password hashing.

Checkout & Order Flow: Secure checkout experience that automatically updates the user's personal dashboard with order history and statuses.

Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

Tech Stack:

a. Frontend

1. React (via Vite)

2. Redux Toolkit (State Management)

3. React Router (Navigation)

4. Lucide React (Icons)

5. Custom CSS (Glassmorphism & Animations)

b. Backend

1. Node.js & Express.js

2. MongoDB & Mongoose (Database & ORM)

3. JSON Web Tokens (JWT) for Authentication

4. Cors & Dotenv
